Supermarket Product Management System (Final Project of Python Course)
Features
 - Add, save, edit and remove products
 - Automatic calculation of total price 
 - Validation for correct input (product, brand, quantity, price, expiration date)
 - GUI built with Tkinter
 - Stores data in an SQLite database

Structure and folder
- product_model file: product class defines product structure
- product_controller file: handles validation and connects GUI to database
- product_dao file: ProductDataAccess manages database operations
- supermarket_view file: Tkinter GUI
